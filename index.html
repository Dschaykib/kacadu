Kacadu

A KAnka CAmpagain DUplicator

This website let's you import an exported Kanka campaign. You could use this as a way to create a backup or to duplicate a complete campaign.

There are only four steps:

  1. Enter a valid personal access token (PAT) for Kanka

    To get a PAT you can follow this link to create one.

  2. Select a campaign ID

    This will be the target of the import. The dropdown shows a list of campaings you are able to create entites on.

  3. Upload the exported zip file

    In the campaign tab on Kanka, there is a submenu where you can export your campaign. The created zip file need to be uploaded here.

  4. Press Start

    After pressing the Start button the entites are evaluated and uploaded onto the choosen campaign. This step might take a while, depending on the size of your campaign due to the API's rate limits.
